About this template

Use this as a base for a portfolio, product landing page, or simple informational site. Swap the text and colors, and you're ready to go.

Built with basics HTML, CSS, and vanilla JavaScript only. Easy to understand and modify.
Starter friendly A good learning project if you're new to front-end development.

How to use this:
  1. Save this file as index.html.
  2. Open it in your browser (double-click or drag into a tab).
  3. Edit the text, colors, and layout in your code editor.
  4. Upload it to any static hosting provider when you're ready.

You can also split the CSS and JavaScript into separate files (style.css, script.js) and link them if your project grows.
